# 変更履歴

このプロジェクトの主な変更はこのファイルに記録します。

## [Unreleased]

## [1.1.0] - 2026-04-05

### 追加
- `--match-path` モード: A と B でパスが同じかつ内容も同じファイルを B から削除する
- `--include-all` フラグ: `sfd index` のデフォルト除外 (`.git` 等) を無効化する
- MIT ライセンス
- GitHub Actions: CI ワークフロー (main へのプッシュ/PR で `go test` を実行)
- GitHub Actions: リリースワークフロー (バージョンタグでクロスプラットフォームバイナリをビルド)
- 英語ドキュメント。日本語版は `*.ja.md` として残す
- README に `go install` の手順を追加

### 変更
- ドキュメントおよびテストで "A/B比較モード" を "集合モード" に改名

### 修正
- `sfd plan` で `--a` と `--b` に同じファイルを指定した場合にエラーになるよう修正

## [1.0.0] - 初回リリース

- `sfd index`: ディレクトリのチェックサムインデックスを作成・更新する
- `sfd plan`: 削除プランを生成する (集合モード・自己重複検出モード)
- `sfd apply`: 削除プランを実行またはドライランする
- インデックスおよびプランファイルに JSONL フォーマットを採用
